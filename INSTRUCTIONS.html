üé® Color Customization

All colors are defined as CSS variables in styles.css at the very top of the file. Modify these to change the entire site's color scheme:

Default Color Palette

Variable Name Default Value Usage
--primary #00d4ff (Cyan) Main accent color
--secondary #ff006e (Pink) Secondary accent
--accent #00ff88 (Green) Success/highlight color
--accent-purple #a100f2 (Purple) Gradient color
--dark-bg #0a0e27 (Dark) Main background

How to Change Colors

1 Open styles.css

2 Find the :root section at the top

3 Replace the hex color values with your new colors

4 Save the file and refresh your browser

:root {
    --primary: #00d4ff;        /* Change to your color */
    --secondary: #ff006e;      /* Change to your color */
    --accent: #00ff88;         /* Change to your color */
    /* ... more colors ... */
}
üí° Tip: Use a color picker tool like color-hex.com to find perfect hex color values for your brand.

‚úèÔ∏è Content Modification

All text content is stored in index.html. Here's how to modify different sections:

Changing the Page Title

Located in the <head> section:

<title>CoinQuest - Play. Save. Win.</title>
<meta name="description" content="Your new description here">

Modifying Main Headings

Search for text in index.html to find and edit section titles:

Editing Section Content

  • Find the section by searching for its name (e.g., "Features", "Community")
  • Locate the text content within that section
  • Edit the text directly
  • Maintain the HTML structure (don't remove tags)
‚ö†Ô∏è Warning: Be careful not to accidentally delete HTML tags like <div>, <span>, or <class> attributes, as this will break the styling.

‚ú® Animation Customization

Animation Speed

Animations are controlled by CSS transitions. To make things slower or faster:

/* Find this in styles.css and adjust the time value */
transition: all 0.3s ease;  /* 0.3 seconds - change this number */

Common values: 0.1s (very fast), 0.3s (fast), 0.6s (medium), 1s (slow)

Easing Functions

Control how animations feel:

Easing Type Effect
ease Smooth default easing
ease-in Slow start, fast end
ease-out Fast start, slow end
linear Constant speed

Disabling Animations

To remove animations, set the transition to 0s:

transition: all 0s ease;  /* Animations disabled */
‚ö†Ô∏è Note: Scroll-triggered animations use JavaScript. To modify these, edit the script.js file and look for the IntersectionObserver section.

üõ†Ô∏è Additional Customizations

Changing Fonts

Fonts are imported from Google Fonts in the <head>:

<link href="https://fonts.googleapis.com/css2?family=Poppins:wght@300;400;600;700;800&family=Orbitron:wght@400;700;900&display=swap" rel="stylesheet">

Visit Google Fonts to choose different fonts and update the URL.

Modifying Favicon

Replace favicon.svg with your own SVG file:

<link rel="icon" href="favicon.svg" type="image/svg+xml">

Keep it as SVG for best quality and scalability.

Container Width

Adjust the max-width of the page content in styles.css:

.container { max-width: 1200px; /* Change this value */ margin: 0 auto; padding: 0 20px; }

Responsive Breakpoints

The page has responsive design for different screen sizes. Media queries are at the bottom of styles.css:

  • @media (max-width: 1024px) - Tablets
  • @media (max-width: 768px) - Small tablets/large phones
  • @media (max-width: 480px) - Small phones
